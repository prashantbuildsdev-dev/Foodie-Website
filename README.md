
# Foodie 🍔

Foodie is a fully responsive food ordering website built using HTML, CSS, and JavaScript.

## Features
- Responsive design for mobile and desktop
- Dynamic product loading using JSON
- Add to Cart functionality
- Cart quantity management (+/-)
- Automatic total price calculation
- Mobile hamburger menu
- Customer review slider using Swiper.js
- Modern and clean UI/UX

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)
- Swiper.js
- Font Awesome

## Project Highlights
- Interactive shopping cart system
- Dynamic DOM manipulation
- Responsive layout with Flexbox
- Smooth animations and transitions
- JSON-based product management
